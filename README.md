# 4-dim-triangulations

This code was written by Ahmad Issa for his MPhil thesis titled "Triangulating Cappell-Shaneson homotopy 4-spheres".

Unfortunately, this code wasn't organized in a nice way to be shared but hopefully it's still useful. Note that this is old Python code written for Python 2. The code itself is self-contained and I bet a Python 2 to Python 3 conversion tool would work. As an entry point to the code, you can take a look at cappell_shaneson_example.py and driver.py.

Note that the files in data/gluck_twist_pieces are important if you want to do Gluck twists.
